## Development

This site is built with [Hugo](https://gohugo.io/).

```bash
# Install Hugo
sudo apt install hugo

# Build locally
hugo --minify

# Serve locally (http://localhost:1313)
hugo server
```

## Deployment

Automatically deployed to GitHub Pages on push to `main` via `.github/workflows/hugo.yml`.
